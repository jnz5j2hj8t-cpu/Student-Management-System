# Student Management System

A simple student management system built with Python.

This project allows users to manage students and their IELTS scores.

## Features

- Show students
- Add student
- Remove student
- Search student
- Show average IELTS score
- Update IELTS score
- Show statistics
- Save to file
- Load from file
